# charm_

An ecosystem for writing [[cli]] apps in [[go]].

There are libraries for [[UI]] components, forms, styling, ssh, animations and logging.

And more apps built on-top of this ecosystem.

## My favourites

I tried using the markdown renderer `glow`, but found a tighter [[neovim]] integration better.

There is [vhs](https://github.com/charmbracelet/vhs), a asciinema alternative for terminal recordings, and [Skate](https://github.com/charmbracelet/skate), a terminal key-value store.
