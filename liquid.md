# Liquid

A templating language, which uses `{{ objects }}`, to specify templated sections.

It offers tags, which are objects with the `%` character that control flow around the template, e.g. `{{ %if true }}`

Another feature are filters, which are pipelines steps inside objects, e.g. `{{ object | lowercase }}`
