# Microfeatures I Love in Blogs and Personal Websites

This blog post has a collection of tiny features recommended by its author.

Also, this blog has fantastic typography.

## Recommended features

- Sidenotes
- Showing page progress
- linkable headings
- series, grouped posts
- dialogues
- markers for external links
- fancy codeblocks with symbol definition lookup
- rss feeds
