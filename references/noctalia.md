# Noctalia - Wayland Shell

An opinionated desktop shell, built on top of [[quickshell]].

By using quickshell, it avoids common other desktop tooling, like `dunst`, `hyprlock`, `rofi`.
It only has [[ui]] components integrated.
Any functionality, like network management, clipboard, brightness control, file picking dialogs are still handled by external tools.
