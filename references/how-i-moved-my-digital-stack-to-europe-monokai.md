# How I Moved My Digital Stack to Europe — Monokai

A blog post about moving an exosystem and various other tech solutions to european alternatives.

- google analytics -> self-hosted [matomo][matomo]
- gmail -> proton mail
- 1password -> proton pass
- digitalocean -> scaleway
- aws s3 -> scaleway
- backblaze -> ovhcloud
- twilio -> [lettermint]

and more...

Some stuff was not moved, for various reasons.

- cloudflare: basically anything runs on cloudflare, so not having a website on cloudflare would not matter.
- github: selfhosting git is possible, but discovery still is best on github
- ai coding assistance: claude code is king
