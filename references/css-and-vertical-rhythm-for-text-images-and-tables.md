# [CSS](css) and vertical rhythm for text, images, and tables

A blog post about the proper return-to-text from injected elements, like images or tables.

This post recommends using the `lh` and `rlh` units to relate to line-height and root-element line-height.

It also has a lovely font, with sans-serif for the headings and serif for the flowing text.

## Details

Images should be kept to multiples of line-height, and reponsive images should have padding to get them to the nearest line-height.

There are also line-height presets for various [[html]] elements.
