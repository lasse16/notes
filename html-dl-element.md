# HTML: `<dl>` element

The description list element, used for marking a list of `<dt>` and `<dd>` pairs.

Given the names, it was intended for glossaries and metadata, but it works also for any list of key-value pairs.

See the [[references/on-the-andlt-dlandgt-ben-myers]] post for examples.
