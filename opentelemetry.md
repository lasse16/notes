# OpenTelemetry

This is an open-source framework for observability.

It combines a set of APIs, collector agents and libraries for collecting logs and traces across multiple services.

It is more of a language and delivery system than a storage backend or visualization tool.

Three parts of its language are `Traces`, `Metrics` and `Logs`.
There are different SDKs for multiple languages, which just follow this pattern and are hence compatible with OpenTelemetry.
