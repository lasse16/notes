# CSS

A language describing properties of elements in structured markup languages.

Commonly used to describe [[html]] elements on websites, but can also be used to design display of [[xml]], as in [[svg]] files.

It is more a description language for designers than a proper "programming" language and does not have any control flow keywords.

To fix this, there are various [[pre-processors]], which add modules, custom shortcodes, nested constructs and more.
One of the most common [[pre-processors]] is [[sass]], which is integrated in the build-step of [Static-site generators](static-site-generator), like [[jekyll]].
