# HTTP: Hypertext transfer protocol

The basic scheme for accessing websites.

The protocol has different methods, which are sent to a remote server.

These methods include `GET`, `POST`, `DELETE`, `PATCH`.

Additionally, headers, a body and cookies can also be used to include data into a request.
