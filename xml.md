# XML: Extensible Markup language

A data language, which is famous for its amount of tags.

It is the basic schema and parent of [[html]], [[rss]], [[svg]] and many more.

The example structure is:

```xml
<?xml version="1.0" encoding="UTF-8"?>

<opening_tag> text </closing_tag>
<empty_tag />

<multi_line_tag>
    <nested_tag />
</multi_line_tag
```
