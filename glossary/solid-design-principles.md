# SOLID design principles

A set of guidelines for [Object Oriented Programming](object-oriented-programming).

These guidelines are [Single Reponsibility](single-reponsibility), [Open-Closed principle](open-closed), [liskov substitution](liskov-substitution), [interface segregation](interface-segregation) and [Dependency injection](dependency-injection).
