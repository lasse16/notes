# Desktop environments

A collection of tools to supply a classic [[gui]] desktop metaphor for computers.

While operating systems, like Windows or MacOS, come with pre-built desktop environment, on Linux one can often choose a desktop environment.
There are certain projects associated with desktop environments, like KDE, GNOME or Cinnamon, which each ship their own desktop environment.

A desktop environment mainly consists of a window managing system, and a set of widgets, like network control, brightness, a shutdown menu and more.

It is possible to build your own desktop environment, by supplying a tool for each widget you want and binding them together with a custom window manager, like [[niri]].
