# Static-site generators

A tool, which generates website, mostly static [[html]] and [[css]] files, from a set of source files.

These source files are most commonly [[markdown]] files.

## Common options

- [Quartz][quartz]:
It is targeted directly at note-wikis, rendering [[markdown]] files as websites.
This is a [Node.JS](node-js) application, which is extended via [[typescript]].
