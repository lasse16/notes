# KCL

New language that transpiles into yaml, but also allows for Schemas and Data-validation.

It has a strong [[cloud-native]] focus, with pre-defined schemas for all Kubernetes resources.

Templating, Schemas, Editor-integration, YAML Generation.

## Tooling

- LSP integrated in KCL binary
- [Neovim Plugin](https://github.com/kcl-lang/kcl.nvim)
