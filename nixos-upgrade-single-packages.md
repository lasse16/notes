# [NixOS](nixos): upgrade single packages

[NixHub][nixhub] offers a lookup on nixpkg hashes based on program versions, e.g. get a nixpkgs SHA based on which Neovim version you want.

If setup correctly, you can pass in a different nixpkgs version, and select your program from that nixpkgs version.
