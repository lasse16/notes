# Markdown

A markup language, commonly used for simple text structuring.

It is used in many places due to its simplicity.
These places include source files for websites, text descriptions of images, emails, in-line editors on webpages and many more.

There are multiple dialects, each introducing different elements, but the default spec is [CommonMark][common_mark].
Another famous dialect is [[gfm]], which includes a table syntax.
