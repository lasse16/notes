# Bash: Consistent tooling

A blog post collection common design questions and idioms for [Bash](bash) tools.

use `-V | --version` for version printing.

use `-h | --help` for help printing

use `-u | --username` and `-p | --password` for user authentication

use `-v | --verbose` to increase the output verbosity
